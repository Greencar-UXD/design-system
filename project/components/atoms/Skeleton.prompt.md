Skeleton — loading placeholder with a subtle shimmer (stills under reduced-motion).

```jsx
import { Skeleton } from './components/display/Skeleton';

<Skeleton variant="circle" width={40} height={40} />
<Skeleton variant="text" width="60%" />
<Skeleton height={120} />
```

- **variant**: `block` · `text` · `circle`. Size with `width`/`height`.
